### BestBuy Autocomplete

Dataset used: [https://github.com/BestBuyAPIs/open-data-set](https://github.com/BestBuyAPIs/open-data-set)

#### Requirements
- Go 1.11+
- Java 11
- Docker-engine v20.10.2
- Terraform v0.14.5

#### Instructions (docker)

1. terraform init
2. terraform apply
3. go run go/main.go
4. [visit web page at localhost:3000](http://localhost:3000)
5. terraform destroy
